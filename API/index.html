Plant Project API - Routes


Root: https://plantprojectapi.azurewebsites.net/

Plant

  • GET all plants: /plant/
  • POST new plant: /plant/
  • GET plant by id: /plant/id/{plantId}
  • GET plant by name: /plant/name/{name}
  • DELETE plant by id /plant/id/{plantId}
  • DELETE plant by name /plant/name/{name}
  • UPDATE plant by id /plant/id/{plantId}
  • UPDATE plant by name /plant/name/{name}

Air Humidity

  • GET all air humidity data: /airhumidity/
  • POST new air humidity data: /airhumidity/
  • GET air humidity by id: /airhumidity/id/{humidityId}
  • GET air humidity by plantid: /airhumidity/plantId/{plantId}
  • GET latest air humidity by plantid: /airhumidity/latest/{plantId}
  • GET top X air humidity by plantid: /airhumidity/latest/{plantId}/{amount}
  • DELETE air humidity by id /airhumidity/id/{humidityId}
